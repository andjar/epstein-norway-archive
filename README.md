## Status

Prosjektet er pågående og langt fra ferdig. Teksten er primært hentet fra https://github.com/promexdotme/epstein-justice-files-text/ og må kryss-sjekkes mot original-kildene.

Tidspunkt er en utfordring på grunn av ulike tidssoner. VG har laget en oversikt som kanskje kan brukes til avstemming: https://www.vg.no/nyheter/i/Ar3wW5/mette-marit-og-epstein-slik-var-dialogen

## Hvordan bidra?

Dette prosjektet er bygget på https://tiddlywiki.com/ og mailene ligger som tiddlers her på github: https://github.com/andjar/epstein-norway-archive/tree/main/tiddlers

Her er et eksempel på strukturen:

```
title: $:/mails/892477cb-02a7-4219-bc41-bafdfbe72f72
from: [[Jeffrey Epstein]]
to: [[Kronprinsesse Mette-Marit]]
subject: Re: Re:
messageid: 892477cb-02a7-4219-bc41-bafdfbe72f72
tags: Mail [[Jeffrey Epstein]] [[Kronprinsesse Mette-Marit]]
time: 20111024062700000
source: [[EFTA01772997.pdf]]
parent: b0c41b98-efd4-43fd-8d9d-29ffb273cfc9

Sweetness is a better translation ..thanks 
Sent from my iPad
```

Du kan klone dette github-prosjektet og korrigere materialet eller f.eks. forbedre brukergrensesnittet. Du kan også gjøre dette fra nettleseren. Det er en blyant man kan trykke på for å redigere innholdet, når dette er gjort, må du trykke på tilhørende pil, velge "export tiddler" og deretter "TID text file". Det vil gi deg riktig format, som du kan pushe tilbake til dette prosjektet.
